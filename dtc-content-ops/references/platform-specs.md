# Platform Specifications Reference

Quick reference for social media platform constraints and best practices.

## Character Limits

| Platform | Hard Limit | Recommended Max | Notes |
|----------|-----------|-----------------|-------|
| X (Twitter) | 280 | 240 | Leave room for engagement; URLs count as ~23 chars |
| Threads | 500 | 400 | Conversational, not a blog post |
| Instagram Feed | 2,200 | 300-500 | First 125 chars visible before "more" |
| Instagram Story | N/A | 8-15 words | Text overlay on image/video |
| Instagram Reel | 2,200 | 150-300 | Short, hook-driven |
| TikTok | 2,200 | 100-150 | Aggressively truncated in feed |
| Shopify Blog | No limit | 600-1,200 words | SEO-optimized articles |
| Blogger | No limit | 600-1,200 words | SEO-optimized articles |

## Image Aspect Ratios

| Platform | Feed | Story/Reel | Recommended Resolution |
|----------|------|-----------|----------------------|
| X | 1:1 or 16:9 | N/A | 1200x1200 or 1200x675 |
| Instagram Feed | 4:5 | N/A | 1080x1350 |
| Instagram Story/Reel | N/A | 9:16 | 1080x1920 |
| Threads | 1:1 | N/A | 1080x1080 |
| TikTok | N/A | 9:16 | 1080x1920 |
| Blog | 16:9 | N/A | 1200x675 |

## Hashtag Best Practices

| Platform | Count | Placement |
|----------|-------|-----------|
| X | 0-2 | Inline or end of post |
| Instagram Feed | 5-15 | End of caption or first comment |
| Instagram Reel | 3-5 | In caption |
| Threads | 0-3 | Inline only |
| TikTok | 3-5 | End of caption |
| Blog | 0 | Use meta tags instead |

## Posting Frequency Limits

| Platform | Daily Limit | Monthly Limit | Notes |
|----------|------------|--------------|-------|
| X | ~50 | 500 (free) | Rate limited, not hard capped |
| Instagram | 25 per account | ~750 | Includes all content types |
| Threads | 250 per account | ~7,500 | Per-account limit |
| TikTok | 5 videos/day | ~150 | Per-account limit |
| Shopify Blog | No limit | N/A | Practical limit: 1-2/day for SEO |

## Media Type Support

| Platform | Image | Video | Carousel | Text-only |
|----------|-------|-------|----------|-----------|
| X | Yes | Yes (2:20 max) | No | Yes |
| Instagram Feed | Yes | Yes (90s) | Yes (10 slides) | No |
| Instagram Story | Yes | Yes (60s) | Auto-split | No |
| Instagram Reel | No | Yes (90s) | No | No |
| Threads | Yes | Yes | Yes (10) | Yes |
| TikTok | Yes (photo) | Yes (10min) | Yes (slideshow) | No |
